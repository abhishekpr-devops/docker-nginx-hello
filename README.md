# Docker Nginx Hello - Version 2

A simple Docker project that serves a custom HTML page using Nginx through a custom Docker image.

Files:
- index.html
- Dockerfile
- run.sh
- README.md

Build and run:
./run.sh

Open:
http://localhost:8090

Verify:
curl localhost:8090

What I learned:
- How to create a Dockerfile
- How to build a custom Docker image
- How to run a container from my own image
- How port mapping works
